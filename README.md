# 毎朝の運勢 0.9.0-rc5

公開候補版 RC5 です。

## 今回の修正
- 起動画面の「おはようございます」を約3秒表示
- フェード時間も約0.65秒へ延長
- 「今日のひとこと」を26pxへ拡大
- 「今日のひとこと」カードの余白も拡大
- 「詳しい運勢を見る」ボタンを完全削除
- 仕事運・恋愛運・金運・健康運を常時表示
- CSS・JavaScript・Manifestへバージョン付きURLを付与
- Service Workerを作り直し、旧キャッシュをすべて削除
- HTMLはネットワーク優先で読み込み、更新を反映しやすく変更

## GitHub Pagesへの反映
ZIP内のファイルをリポジトリ直下へすべて上書きしてください。

## 初回確認
GitHub Pagesへ反映後、Safariで一度ページを閉じて開き直してください。
ホーム画面追加済みの場合も、アプリを完全に終了してから再起動すると確実です。
