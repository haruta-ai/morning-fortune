# 毎朝の運勢 v0.8.0

v0.7.1を土台に、既存のデザインと機能を維持したまま履歴画面を再設計した正式アップデート候補です。

## 主な変更
- 履歴画面上部の月間カレンダーを大型化
- iPhoneで押しやすい48px以上の月移動ボタンと、大きな日付セル
- カレンダーの日付タップをイベント委譲方式に変更し、再描画後も安定して反応
- 選択した日の詳細をダイアログではなくカレンダー直下に表示
- 記録がない日もタップ可能にし、「記録なし」を明示
- 今月へ戻る操作を月タイトルに追加
- Service Workerとキャッシュ名をv0.8.0へ更新
