# 毎朝の運勢 v0.8.0

## 主な変更
- 履歴画面を全面再設計
- 月間カレンダーを履歴画面の最上部へ大型配置
- 日付タップで、その日の運勢・各項目・振り返りを詳細表示
- 下部ナビゲーションの5画面をすべて動作する構成へ統一
- iPhoneのSafe Area、タップ領域、文字サイズ、スクロールを調整
- お気に入り・振り返り・設定をlocalStorageへ保存
- 外部API不使用
- GitHub Pages / PWA向けService Workerを追加

## GitHub Pagesへの更新
このフォルダ内のファイルを、公開中リポジトリの同じ場所へ上書きしてください。

- index.html
- style.css
- script.js
- manifest.webmanifest
- sw.js

古いService Workerのキャッシュが残る場合は、Safariでページを再読み込みするか、
ホーム画面へ追加済みのアプリを一度終了して開き直してください。

## 動作確認項目
1. 下部メニュー5個がそれぞれ開く
2. 履歴画面の前月・翌月・今月ボタンが動く
3. 日付をタップすると詳細が更新される
4. 前月・翌月の日付をタップすると該当月へ切り替わる
5. 今日のお気に入り登録・解除ができる
6. 振り返りを保存し、カレンダーに記録点が表示される
7. 設定の文字サイズ・動きを減らすが反映される
8. iPhoneの下部ナビがホームインジケータに重ならない
