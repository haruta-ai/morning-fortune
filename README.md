# 毎朝の運勢 0.9.0-rc1

公開候補版 RC1 です。

## 主な改善
- 朝焼け・星・光を使った世界観の統一
- iPhone向けの余白、文字サイズ、Safe Area調整
- 起動スプラッシュの短縮と自然化
- 時間帯に応じた挨拶
- 運勢データを7パターンへ拡充
- 今日の注意点を追加
- 使い方ダイアログを追加
- 更新通知とService Worker更新処理を改善
- オフライン状態表示
- その日の運勢を端末内に保存
