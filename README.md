# 毎朝の運勢 0.9.0-rc4

公開候補版 RC4 です。

## 主な改善
- 起動画面の「おはようございます」を約2秒表示
- 「今日のひとこと」を21pxへ拡大し、余白と太さも調整
- 詳しい運勢の開閉式を廃止
- 仕事運・恋愛運・金運・健康運を最初から表示
- 余計な操作を減らし、毎朝すぐ読める構成へ変更
- 項目別カードを少しコンパクト化
- オフライン・更新通知・日付変更処理を継続
